Cara Menambah Admin Baru?

Untuk alasan keamanan tingkat tinggi, penambahan admin baru hanya dapat dilakukan melalui konsol Firebase Authentication oleh pemilik database utama.

1. Buka Firebase Console.
2. Pilih menu Authentication > Users.
3. Klik Add User dan masukkan email serta password untuk admin baru.
